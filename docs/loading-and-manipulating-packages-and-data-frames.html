2.4.5 Recode a text variable

Tidyverse libraries are needed for the following examples.

data("band_members")
kable(head(band_members)) %>% kable_minimal(full_width = F)
name band
Mick Stones
John Beatles
Paul Beatles 
band_members <- band_members %>% mutate(name=recode(name, "Mick"= "m"))
kable(head((band_members))) %>% kable_minimal(full_width = F)
name band
m Stones
John Beatles
